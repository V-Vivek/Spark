
# PySpark DataFrame and SparkSQL Data Types


## Numeric Types

### Integer Types
- **ByteType**: 8-bit signed integer (-128 to 127)
- **ShortType**: 16-bit signed integer (-32,768 to 32,767)
- **IntegerType**: 32-bit signed integer (-2,147,483,648 to 2,147,483,647)
- **LongType**: 64-bit signed integer

### Floating Point Types
- **FloatType**: 32-bit single-precision floating point
- **DoubleType**: 64-bit double-precision floating point

### Decimal Type
- **DecimalType**: Arbitrary-precision signed decimal (default: Decimal(10, 0))

## String Types

- **StringType**: Character string values, with variable length
- **VarcharType(length)**: Variable-length character string with maximum length limit
- **CharType(length)**: Fixed-length character string (padded with spaces if shorter)

### Example: 
```python
spark.sql('CREATE TABLE test(c1 STRING, c2 VARCHAR(10), c3 CHAR(10))')
spark.sql('INSERT INTO test VALUES ("Alice", "Alice", "Alice")')
spark.sql('SELECT * FROM test;').show()
spark.sql('SELECT LENGTH(c1) c1_len, LENGTH(c2) c2_len, LENGTH(c3) c3_len FROM test;').show()

# Output:
+-----+-----+----------+
|   c1|   c2|        c3|
+-----+-----+----------+
|Alice|Alice|Alice     |
+-----+-----+----------+

+------+------+------+
|c1_len|c2_len|c3_len|
+------+------+------+
|     5|     5|    10|
+------+------+------+
```

## Boolean Type

- **BooleanType**: True or False values

## Binary Type

- **BinaryType**: Binary (byte array) data

## Date and Time Types

- **DateType**: Date values (year, month, day)
- **TimestampType**: Timestamp with date and time
- **TimestampNTZType**: Timestamp without timezone (Spark 3.4+)

## Complex Types

- **ArrayType(elementType, containsNull)**: Ordered collection of elements
- **MapType(keyType, valueType, valueContainsNull)**: Key-value pairs (dictionary-like)
- **StructType(fields)**: Complex type with named fields (row structure)

## Other Types

- **NullType**: Represents null values


## Usage Example

```python
from pyspark.sql.types import *
from datetime import date, datetime

schema = StructType([
    StructField("id", IntegerType(), False),
    StructField("name", StringType(), True),
    StructField("salary", DoubleType(), True),
    StructField("is_active", BooleanType(), True),
    StructField("data", BinaryType(), True),
    StructField("join_date", DateType(), True),
    StructField("last_login", TimestampType(), True),
    StructField("tags", ArrayType(StringType()), True),
    StructField("properties", MapType(StringType(), StringType()), True)
])

data = [
    (1, "Alice", 70000.0, True, bytearray(b'\x00\x01'), 
     date(2020, 1, 15), datetime(2023, 6, 1, 10, 0, 0), 
     ['tag1', 'tag2'], {'key1': 'value1'}),
    (2, "Bob", 80000.0, False, bytearray(b'\x02\x03'), 
     date(2019, 3, 22), datetime(2023, 6, 2, 11, 30, 0), 
     ['tag3'], {'key2': 'value2'})
]
df = spark.createDataFrame(data, schema)
df.show(truncate=False)
df.printSchema()

# Output:
+---+-----+-------+---------+-------+----------+-------------------+------------+----------------+
|id |name |salary |is_active|data   |join_date |last_login         |tags        |properties      |
+---+-----+-------+---------+-------+----------+-------------------+------------+----------------+
|1  |Alice|70000.0|true     |[00 01]|2020-01-15|2023-06-01 10:00:00|[tag1, tag2]|{key1 -> value1}|
|2  |Bob  |80000.0|false    |[02 03]|2019-03-22|2023-06-02 11:30:00|[tag3]      |{key2 -> value2}|
+---+-----+-------+---------+-------+----------+-------------------+------------+----------------+

root
 |-- id: integer (nullable = false)
 |-- name: string (nullable = true)
 |-- salary: double (nullable = true)
 |-- is_active: boolean (nullable = true)
 |-- data: binary (nullable = true)
 |-- join_date: date (nullable = true)
 |-- last_login: timestamp (nullable = true)
 |-- tags: array (nullable = true)
 |    |-- element: string (containsNull = true)
 |-- properties: map (nullable = true)
 |    |-- key: string
 |    |-- value: string (valueContainsNull = true)
```

## Enforcement of Data Types

**DataFrame creation:** Not strictly enforced; data is cast to specified types when possible.  
**SparkSQL:** Enforced during table creation and data insertion; type mismatches result in errors.
