
# PySpark Date Functions

- **current_date**: Returns the current date as a date column.
    - Syntax: `current_date()`

- **current_timestamp**: Returns the current timestamp as a timestamp column.
    - Syntax: `current_timestamp()`

- **next_day**: Returns the first date which is later than the value of the date column that is on the specified day of the week.
    - Syntax: `next_day(date, dayOfWeek)`

- **last_day**: Returns the last day of the month which the given date belongs to.
    - Syntax: `last_day(date)`

- **dayofweek**: Extracts the day of the week as an integer (1=Sunday, 7=Saturday).
    - Syntax: `dayofweek(date)`

- **dayofmonth**: Extracts the day of the month as an integer (1-31).
    - Syntax: `dayofmonth(date)`

- **dayofyear**: Extracts the day of the year as an integer (1-366).
    - Syntax: `dayofyear(date)`

- **weekofyear**: Extracts the week of the year as an integer (1-53).
    - Syntax: `weekofyear(date)`

- **second**: Extracts the seconds of a given timestamp as an integer (0-59).
    - Syntax: `second(timestamp)`

- **minute**: Extracts the minutes of a given timestamp as an integer (0-59).
    - Syntax: `minute(timestamp)`

- **hour**: Extracts the hour of a given timestamp as an integer (0-23).
    - Syntax: `hour(timestamp)`

- **month**: Extracts the month as an integer (1-12).
    - Syntax: `month(date)`

- **quarter**: Extracts the quarter of the year as an integer (1-4).
    - Syntax: `quarter(date)`

- **year**: Extracts the year as an integer.
    - Syntax: `year(date)`

- **months_between**: Returns the number of months between two dates.
    - Syntax: `months_between(date1, date2, roundOff=True)`

- **date_add**: Adds a specified number of days to a date.
    - Syntax: `date_add(date, days)`

- **date_sub**: Subtracts a specified number of days from a date.
    - Syntax: `date_sub(date, days)`

- **add_months**: Adds a specified number of months to a date.
    - Syntax: `add_months(date, months)`

- **datediff**: Returns the number of days between two dates.
    - Syntax: `datediff(date1, date2)`

- **date_trunc**: Truncates a timestamp to the specified unit (year, month, day, hour, etc.).
    - Syntax: `date_trunc(format, timestamp)`

- **date_format**: Converts a date/timestamp to a string using the specified format.
    - Syntax: `date_format(date, format)`

- **unix_timestamp**: Converts a time string or timestamp to Unix timestamp (seconds since epoch).
    - Syntax: `unix_timestamp(timestamp, format)`

- **to_timestamp**: Converts a string to a timestamp using an optional format.
    - Syntax: `to_timestamp(col, format=None)`

- **from_unixtime**: Converts Unix timestamp to a string representation of the timestamp.
    - Syntax: `from_unixtime(timestamp, format='yyyy-MM-dd HH:mm:ss')`

- **from_utc_timestamp**: Converts a UTC timestamp to a timestamp in a specified timezone.
    - Syntax: `from_utc_timestamp(timestamp, tz)`

- **to_date**: Converts a string or timestamp to a date using an optional format.
    - Syntax: `to_date(col, format=None)`
