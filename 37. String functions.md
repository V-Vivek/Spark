
# PySpark DataFrame String Functions

## String Manipulation Functions

| Function | Description |
|----------|-------------|
| `split(str, pattern)` | Splits a string column into an array of substrings based on a delimiter pattern. |
| `length(col)` | Returns the length (number of characters) of a string column. |
| `lower(col)` | Converts all characters in a string column to lowercase. |
| `upper(col)` | Converts all characters in a string column to uppercase. |
| `initcap(col)` | Converts the first letter of each word in a string to uppercase and the rest to lowercase. |
| `ltrim(col)` | Removes leading whitespace characters from the left side of a string. |
| `rtrim(col)` | Removes trailing whitespace characters from the right side of a string. |
| `trim(col)` | Removes leading and trailing whitespace characters from both sides of a string. |
| `lpad(col, len, pad)` | Left-pads a string column with a specified character to reach a given length. |
| `rpad(col, len, pad)` | Right-pads a string column with a specified character to reach a given length. |
| `reverse(col)` | Reverses the order of characters in a string column. |
| `repeat(col, n)` | Repeats a string column n times and returns the concatenated result. |
| `hex(col)` | Converts a binary or integer column to its hexadecimal string representation. |
| `concat(*cols)` | Concatenates multiple string columns together without any separator. |
| `concat_ws(sep, *cols)` | Concatenates multiple string columns together with a specified separator between them. |
| `substring(str, pos, len)` | Extracts a substring from a string column starting at a position for a specified length. |
| `substring_index(str, delim, count)` | Returns the substring before count occurrences of the delimiter. |
| `instr(str, substr)` | Returns the position (1-based index) of the first occurrence of a substring within a string. |
| `locate(substr, str, pos)` | Returns the position of the first occurrence of a substring, optionally starting from a specified position. |
| `translate(srcCol, matching, replace)` | Translates characters in a string by replacing characters in the matching set with corresponding characters in the replace set. |
| `overlay(src, replace, pos, len)` | Replaces a portion of a string with another string starting at a specified position for a given length. |
